HTML:

I've learnt some tags which are lesser known and have used them in my projects. The art of learning new Tags in HTML is obviously by implementing them in your projects.
Few Tags:

  1. 1.'slider' Tag : Can be used to implementing the Slider into an HTML
  2. 2.'abbr' Tag: In HTML is used to define the abbreviation or short form of an element
  3. 3.'b' Tag: is used to specify the bold text without any extra importance
  4. 4.'bdi' Tag: This tag refers to the Bi-Directional Isolation. It differentiates a text from other text that may be formatted in a different direction. This tag is used when a user-generated text with unknown directions
  5. 5. 'details' Tag: specifies additional details that the user can open and close on demand
  6. 6.'mark' Tag: is used to highlight the some text part inside of another element such as paragraph, for reference or any notation purpose. This is a newly added tag and introduced in HTML5. In most of the browser, the text contained with 'mark' tag renders with the yellow background, but it can be changed using CSS background-color property
  7. 7.'option' Tag:defines an option in a select list
  8. 8.'param' Tag:used to define a parameter for plug-ins which is associated with object element. It does not contain the end tag
  9. 9.'output' Tag: used to represent the result of a calculation (like one performed by a script).
