HTML- Hyper Text Markup Language:

Here are some tags which are lesser known and have used them in my projects. The art of learning new Tags in HTML is by implementing them in your projects.
Few Tags:

  1. 1.'slider' Tag : Can be used to implementing the Slider into an HTML
  2. 2.'abbr' Tag: In HTML is used to define the abbreviation or short form of an element
  3. 3.'b' Tag : is used to specify the bold text without any extra importance
  4. 4.'bdi' Tag: This tag refers to the Bi-Directional Isolation. It differentiates a text from other text that may be formatted in a different direction. This tag is used when a user-generated text with unknown directions
  5. 5. 'details' Tag: specifies additional details that the user can open and close on demand
  6. 6.'mark' Tag: is used to highlight the some text part inside of another element such as paragraph, for reference or any notation purpose. This is a newly added tag and introduced in HTML5. In most of the browser, the text contained with 'mark' tag renders with the yellow background, but it can be changed using CSS background-color property
  7. 7.'option'Tag:defines an option in a select list
  8. 8.'param' Tag:used to define a parameter for plug-ins which is associated with object element. It does not contain the end tag
  9. 9.'output' Tag: used to represent the result of a calculation (like one performed by a script).


CSS- Cascading Style sheet:

CSS allows you to create great-looking web pages.They specify how documents are presented to users-how they are styled, laid out,etc.
Some properties which are widely used:

  1. 1.display-flex: This property will align your child elements one next to each other in a horizontal row
  2. 2.margin:This property will determine the distance between certain elements
  3. 3.padding: Padding influences the inside of our element.
  4. 4.background color: This property is used to apply the background color to the webpage or any selected element. The color varies in the RGB and RGBA grade.
  5. 5.Color: this property is basically the color of the text content of the web page.
  6. 6.Font-family:Font family is the type of font you want to use it in your text content of the web page. It also includes Font-size property.
  7. 7.hover: this will add any effects to a certain element on hover (:hover).


JavaScript:

JavaScript is a lightweight, interperted, or on-time compiled programming language with first-class functions.It is commonly used for creating interactive web pages and applications.Some basic details about JavaScript for beginner include:

  1. 1.Syntax: JavaScript syntax is similar to other programming languages such as C++ or Java. Statements are written in a sequence and are enclosed in curly braces { }. Semicolons ; are used to separate statements.
  2. 2.Variables: Variables in JavaScript are used to store values such as numbers or text. They are declared using the var, let, or const keywords.
  3. 3.Functions: Functions are used to group a series of statements together to perform a specific task. They can be called at any point in the code.
  4. 4.Objects: Objects are collections of properties and methods. Properties are the characteristics of an object, while methods are the functions that can be performed on an object.
  5. 5.DOM: The Document Object Model (DOM) is a way of representing HTML and XML documents as objects that can be manipulated with JavaScript. This allows for dynamic content and interaction with web pages.
  6. 6.Events: JavaScript can respond to events such as mouse clicks, key presses, and page loads. These events can trigger functions to perform specific actions.
  7. 7.Libraries and Frameworks: There are many libraries and frameworks available in JavaScript to help with common tasks and make coding easier. Examples include jQuery, React, and Angular.
  8. 8.Debugging: Debugging is an important part of coding in JavaScript. Browsers have built-in tools such as the console for viewing errors and debugging code.
  9. 9.ES6: ECMAScript 6 (ES6) is the latest version of JavaScript and introduces many new features such as arrow functions, template literals, and classes. It is important for beginners to be familiar with these new features as they become more widely used in modern JavaScript development.
